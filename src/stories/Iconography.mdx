## Usage

All icons are exported from the `icons` module and accept the following props:

- `size`: `string` - Controls icon size (default: "24")
- `className`: `string` - Additional CSS classes
- `style`: `React.CSSProperties` - Inline styles

### Example Usage

```tsx
import { IconHome, IconCart, IconHeart } from 'icons'

// Basic usage
<IconHome />

// With size
<IconCart size="32" />

// With custom styling
<IconHeart size="40" className="text-red-500" />
```
